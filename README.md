# Steps to run locally
Make sure you have python3 and virtualenv installed 
step 1 
```
git clone https://github.com/guptaautkarsh/flask-backend.git
```
step 2
```
cd flask-backend
```
step 3
```
python3 -m venv venv
```
step 4
```
source venv/bin/activate
```
step 5
```
pip install -r requirements.txt
```
step 6
```
python3 run.py
```
